Introducción

En este proyecto, diseñé y desarrollé un sitio web para una agencia de efectos visuales (VFX). Comencé con una investigación exhaustiva de usuarios y análisis competitivo para obtener información sobre el público objetivo y los estándares de la industria. Luego creé una arquitectura de información integral y un sistema de diseño cohesivo, lo que resultó en un sitio web que destaca el trabajo de vanguardia de la agencia y ofrece una experiencia intuitiva y visualmente atractiva para los usuarios.

Investigación

Empiezo el proyecto realizando investigación. El objetivo es entender a fondo el tipo de usuario, las empresas competidoras y el mercado en el que opera la empresa.

Encuesta

Tras analizar el tipo de usuario de Senzo Studio, preparé las siguientes preguntas.

¿Cómo prevé utilizar el sitio web de Senzo Studio en su flujo de trabajo profesional dentro de la industria de los efectos visuales?

¿Puede describir tareas o actividades concretas para las que confiaría en el sitio web de Senzo Studio para mejorar su trabajo?

En su opinión, ¿cuáles son las funcionalidades o características clave que el sitio web de un estudio de VFX web de un estudio de VFX para satisfacer eficazmente sus necesidades?

¿Qué tipos de contenidos o recursos consideraría esenciales para que el sitio web de un estudio de VFX para que sea una herramienta valiosa para los profesionales del sector?

Teniendo en cuenta su papel y responsabilidades en la industria de los VFX, ¿qué herramientas o crees que deberían ser fácilmente accesibles en el sitio web de Senzo Studio para agilizar tus tareas?

Perspectivas

Tras formular las preguntas a los dos usuarios, extraje la siguiente información.
